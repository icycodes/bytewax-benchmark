### CustomFileSource (FixedPartitionedSource)
- **`list_parts()`**: Returns sorted list of `.txt` filenames in the input directory
- **`build_part()`**: Creates `_FileSourcePartition` instances that read lines from a file
- **`_FileSourcePartition`**:
  - Yields `(filename, line_content)` tuples — one line per batch
  - `resume_state` tracks the next line number to read (0-based)
  - On recovery, seeks to the saved line number, skipping already-processed lines
  - `snapshot()` returns the current line position

### CustomFileSink (FixedPartitionedSink)
- **`list_parts()`**: Returns the same sorted filename list as the source
- **`part_fn()`**: Routes items to the correct partition using the filename key's index in the sorted partition list
- **`build_part()`**: Creates `_FileSinkPartition` instances that write lines to output files
- **`_FileSinkPartition`**:
  - `resume_state` tracks the number of lines committed
  - On recovery, truncates the output file to exactly `resume_state` lines to prevent duplicates
  - `write_batch()` appends lines to the file
  - `snapshot()` returns the total lines written (committed + pending)

### Dataflow
- **Input**: `CustomFileSource` reads from `/home/user/bytewax_project/input_data/`
- **Transform**: `op.map("to_upper", ...)` converts each line to uppercase
