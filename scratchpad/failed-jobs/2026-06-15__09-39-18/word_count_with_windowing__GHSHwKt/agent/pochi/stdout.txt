Created `/home/user/myproject/dataflow.py` — a Bytewax dataflow that performs windowed word counting using event time.

**How it works:**

1. **Input**: Reads a JSONL file where each line is `{"time": "ISO8601_string", "word": "string"}`
2. **Parse**: Each line is parsed into a `(word, datetime)` tuple, with the ISO8601 timestamp converted to a UTC-aware `datetime` object
3. **Window**: Uses `EventClock` (extracting timestamps from events) with `TumblingWindower` (1-hour windows aligned to `2023-01-01T00:00:00Z`) and `count_window` to count occurrences per word per window
4. **Output**: Formats results as JSONL lines `{"word": "...", "window_id": "2023-01-01T00:00:00+00:00", "count": N}` and writes to the output file

**Usage:**
```bash
python dataflow.py <input_file> <output_file>
```

**Key implementation details:**
- `EventClock` with `wait_for_system_duration=timedelta(seconds=0)` for batch-style processing
- `TumblingWindower(length=timedelta(hours=1), align_to=datetime(2023, 1, 1, tzinfo=timezone.utc))`
- Window ID integers are converted to ISO8601 start times via `ALIGN_TO + WINDOW_LENGTH * window_id`
- `run_main()` from `bytewax.testing` executes the dataflow in-process
