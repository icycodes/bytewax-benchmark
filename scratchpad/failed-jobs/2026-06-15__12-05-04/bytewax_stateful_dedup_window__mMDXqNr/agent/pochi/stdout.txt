## What was built

**`/home/user/myproject/run.py`** — A Bytewax 0.21 dataflow with:

1. **`FileSource`** reads `input.jsonl` line-by-line.
2. **`op.map("parse")`** parses each JSON line into a `dict` and stores the ISO 8601 timestamp as a `datetime` object in `_ts`.
3. **`op.key_on("key_by_user")`** keys the stream by `user_id`.
4. **`op.stateful("dedup")`** runs one `DeduplicationLogic` instance per `user_id`:
   - **State**: `Dict[event_id, last_emitted_timestamp]`
   - On each item:
     1. **Evict stale entries**: removes any `event_id` whose `last_ts` is more than 10 seconds before the current event's timestamp (prevents memory leaks).
     2. **Deduplication check**:
        - Not seen before → **emit**, record timestamp.
        - `event_ts > last_ts + 10s` (strictly after window) → **emit**, reset clock.
        - `event_ts ≤ last_ts + 10s` (within or at boundary) → **drop**.
5. **`op.map("serialize")`** formats surviving events back to JSON strings (strips internal `_ts`).
6. **`FileSink`** writes to `output.jsonl`.
