## Implementation Summary

**Pipeline structure:**
1. **Input**: Custom `EventsSource` (subclass of `FixedPartitionedSource`) reads JSON lines from `events.json`
2. **Key by user_id**: `op.key_on` groups events by `user_id` string key
3. **Stateful dedup**: `op.stateful_map` maintains a per-user dictionary of `{event_id: timestamp}`:
   - On each event, prunes entries older than 10 seconds from the current event's timestamp
   - If the `event_id` was already seen within 10 seconds → returns `None` (duplicate)
   - Otherwise → records the event and emits it
4. **Filter**: `op.filter` removes `None` values (duplicates)
5. **Unwrap**: `op.map` extracts the event from the `(key, value)` tuple
6. **Serialize & output**: `op.map` converts to JSON, custom `FileSink` writes to `output.json`
