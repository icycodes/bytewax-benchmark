The Bytewax pipeline has been implemented in `/home/user/project/pipeline.py`. Here's a summary of the dataflow:

**Pipeline (`flow`) steps:**

1. **Read input** — Uses `FileSource` to read `input.jsonl` line by line.
2. **Parse events** — Deserializes JSON, extracts `user_id`, `page`, and parses `timestamp` into a timezone-aware `datetime`.
3. **Key by user** — Keys the stream by `user_id` (string), required for stateful windowing operators.
4. **Define clock & windower** — `EventClock` extracts timestamps from events; `SessionWindower(gap=10s)` closes sessions after 10 seconds of inactivity per user.
5. **Fold window** — Accumulates a count of pages visited per session window.
6. **Format output** — Maps `(user_id, (window_id, total_pages))` → `(user_id, json_line)` for the file sink.
7. **Write output** — Writes keyed JSON lines to `output.jsonl` via `FileSink`.

**Run with:**
```
python -m bytewax.run pipeline:flow
```
