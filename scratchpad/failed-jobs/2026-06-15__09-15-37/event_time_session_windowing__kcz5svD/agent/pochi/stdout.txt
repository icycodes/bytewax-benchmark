## What was built

### `pipeline.py` — Dataflow stages

| Stage | Operator | Purpose |
|---|---|---|
| 1 | `op.input` + `FileSource` | Read `input.jsonl` line-by-line |
| 2 | `op.map("parse")` | Parse JSON → `(user_id, event_dict)` keyed tuple |
| 3 | `EventClock` | Extract UTC-aware timestamp from each event for event-time semantics |
| 4 | `SessionWindower(gap=10s)` | Open/close a session window per user on 10 s inactivity |
| 5 | `fold_window` | Count pages per session (`acc + 1` for each event) |
| 6 | `op.map("format")` | Map `(user_id, (window_id, count))` → JSON string |
| 7 | `op.output` + `JsonlSink` | Append each JSON string to `output.jsonl` |

### Custom `JsonlSink`
`FileSink` (built-in) only accepts keyed tuples; a lightweight `DynamicSink` / `StatelessSinkPartition` subclass was implemented to write plain strings directly to the file.
