## Token-bucket implementation

The state passed to `op.stateful_map` is a plain `tuple[float, float]` — `(current_tokens, last_timestamp)` — which is natively picklable and compatible with Bytewax's SQLite recovery:

```python
def token_bucket_mapper(state, event):
    current_ts = event["timestamp"]
    if state is None:                          # first event → full bucket
        current_tokens, last_ts = INITIAL_TOKENS, current_ts
    else:
        current_tokens, last_ts = state
        elapsed = max(0.0, current_ts - last_ts)
        current_tokens = min(MAX_TOKENS, current_tokens + elapsed * REFILL_RATE)

    if current_tokens >= TOKEN_COST:           # allow
        current_tokens -= TOKEN_COST
        output = event
    else:                                      # drop
        output = None

    return (current_tokens, current_ts), output   # state always updated
```
