ALIGN_TO = datetime(2026, 1, 1, 0, 0, 0, tzinfo=timezone.utc)
WAIT_FOR_SYSTEM = timedelta(seconds=0)

@dataclass
class WindowAccumulator:
    readings: List[Tuple[str, float]] = field(default_factory=list)
    n: int = 0
    mean: float = 0.0
    M2: float = 0.0

    def add(self, timestamp, value):
        self.readings.append((timestamp, value))
        self.n += 1
        delta = value - self.mean
        self.mean += delta / self.n
        delta2 = value - self.mean
        self.M2 += delta * delta2
        return self

    @property
    def std_dev(self):
        if self.n < 2: return 0.0
        return math.sqrt(self.M2 / self.n)

    def merge(self, other):
        merged = WindowAccumulator()
        for ts, val in self.readings + other.readings:
            merged.add(ts, val)
        return merged

## What was built

### `pipeline.py` — Dataflow structure

```
FileSource(data.jsonl)
    → filter_map "parse_json"       # parse JSONL, drop blank/malformed lines
    → key_on "key_by_sensor"        # key by sensor_id (string, required by Bytewax)
    → fold_window "sliding_window"  # 60s windows, 20s step, EventClock
    → flat_map "detect_anomalies"   # 3-sigma outlier check per closed window
    → JsonlFileSink(anomalies.jsonl)
```

### Key design decisions

| Concern | Solution |
|---|---|
| **Sliding windows** | `SlidingWindower(length=60s, offset=20s, align_to=ALIGN_TO)` with `EventClock` driven by record timestamps |
| **Window state** | `WindowAccumulator` dataclass using Welford's online algorithm — stores raw readings + running `(n, mean, M2)` |
| **Picklability** | `WindowAccumulator` is a plain `@dataclass` with only primitive fields and a `List[Tuple[str, float]]`; verified with `pickle.dumps/loads` |
| **SQLite recovery** | Recovery dir initialised with `python -m bytewax.recovery ./recovery_dir 1`; pipeline runs with `-r ./recovery_dir -s 1 -b 0` |
| **Output sink** | Custom `JsonlFileSink(DynamicSink)` — avoids the `(key, value)` tuple requirement of `FixedPartitionedSink` so plain JSON strings flow directly to disk |
| **EOF handling** | `EventClock.on_eof()` returns `UTC_MAX`, causing Bytewax to close all open windows when the input file is exhausted |
